# Audio Library

This folder contains the built-in royalty-free audio tracks for D&D Companion.

## Structure

```
audio-library/
├── combat/        # Battle music and combat sounds
├── exploration/   # Adventure and travel music
├── social/        # Town, tavern, and conversation ambiance
├── tense/         # Suspenseful music
├── dramatic/      # Climactic moments
├── tavern/        # Inn and tavern atmosphere
├── forest/        # Outdoor wilderness sounds
├── dungeon/       # Underground and dungeon ambiance
├── ambient/       # General background music
└── effects/       # Sound effects (non-music)
```

## Adding Tracks

1. Place audio files (MP3 or OGG) in the appropriate category folder
2. Run the seed script to update the database: `npm run db:seed`

## Licensing

All built-in audio is royalty-free and suitable for streaming/recording.

### Recommended Sources for Royalty-Free Audio:
- [Incompetech](https://incompetech.com/) - Kevin MacLeod's music
- [Freesound](https://freesound.org/) - Sound effects
- [OpenGameArt](https://opengameart.org/) - Game audio
- [Tabletop Audio](https://tabletopaudio.com/) - TTRPG-specific ambiance
